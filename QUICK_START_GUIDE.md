# 🚀 MindMirror AI - Quick Start Guide

## 📱 How to Use Each Feature

### 1️⃣ **Analysis Page** (Dashboard)
**URL:** `/dashboard`

**What it does:**
- Analyze your current mood and emotions
- Get personalized recommendations
- View emotional scores

**How to use:**
1. Type your thoughts in the text area
2. Click "Analyze" button
3. View your mood, scores, and AI suggestions

---

### 2️⃣ **Journal Page**
**URL:** `/journal`

**What it does:**
- Create and manage journal entries
- Track your emotional journey
- Search and filter entries

**How to use:**
1. Click "New Entry" button
2. Enter title and content
3. Click "Save" (automatic mood analysis)
4. Use search bar to find entries
5. Use filter dropdown to filter by mood
6. Click edit icon to modify entries
7. Click trash icon to delete entries

---

### 3️⃣ **Calendar Page**
**URL:** `/calendar`

**What it does:**
- View mood patterns over time
- See monthly emotional trends
- Track consistency

**How to use:**
1. Navigate months using arrow buttons
2. Look for colored dots on dates (mood indicators)
3. Click any date to see entries for that day
4. View monthly averages at the bottom

**Mood Colors:**
- 🟢 Green = Thriving
- 🔵 Blue = Positive
- 🟣 Purple = Balanced
- ⚪ Gray = Neutral
- 🟡 Yellow = Low Mood
- 🟠 Orange = Stressed
- 🔴 Red = Burnout

---

### 4️⃣ **Chatbot Page**
**URL:** `/chatbot`

**What it does:**
- Chat with AI wellness companion
- Get emotional support
- Receive mood-aware responses

**How to use:**
1. Type your message in the input box
2. Press Enter or click Send button
3. AI analyzes your mood and responds accordingly
4. Chat history is saved automatically

**Tips:**
- Be honest about your feelings
- Ask for coping strategies
- Request motivational messages

---

### 5️⃣ **History Page**
**URL:** `/history`

**What it does:**
- View charts and analytics
- Track emotional trends
- Export your data

**How to use:**
1. View mood trend line chart
2. Check emotion distribution pie chart
3. Analyze weekly patterns
4. Click "Export Data" to download JSON

---

### 6️⃣ **Profile Page**
**URL:** `/profile`

**What it does:**
- Manage your profile
- View statistics
- Track achievements

**How to use:**
1. Click edit icon to modify profile
2. Upload avatar by clicking camera icon
3. Edit username and bio
4. View your statistics:
   - Total entries
   - Current streak
   - Longest streak
   - Average balance
5. See unlocked achievements

---

### 7️⃣ **Reminders Page**
**URL:** `/reminders`

**What it does:**
- Set wellness reminders
- Get browser notifications
- Stay consistent

**How to use:**
1. Click "Enable" to allow notifications
2. Click "New Reminder" to create custom reminder
3. Set time and message
4. Toggle reminders on/off with switch
5. Click "Test" to test notifications

**Default Reminders:**
- 9:00 AM - Morning check-in
- 12:00 PM - Midday check-in
- 6:00 PM - Evening reflection
- 9:00 PM - Gratitude practice

---

### 8️⃣ **Achievements Page**
**URL:** `/achievements`

**What it does:**
- Track your progress
- Unlock badges
- Stay motivated

**How to use:**
1. View unlocked achievements (colored)
2. Check progress on locked achievements
3. See completion percentage
4. Work towards unlocking all 14 achievements

**Achievement Categories:**
- 📝 Entry-based (1, 10, 50, 100 entries)
- 🔥 Streak-based (3, 7, 30, 100 days)
- 😊 Mood-based (70%+, 85%+ balance)
- 🌅 Time-based (early bird, night owl)
- 💬 Engagement (chatbot messages)

## 💾 Data Storage

**All data is stored locally in your browser:**
- Journal entries
- Chat history
- Profile settings
- Reminders
- Analysis history

**To backup your data:**
1. Go to History page
2. Click "Export Data"
3. Save the JSON file

**Note:** Clearing browser data will delete all entries!

## 🆘 Troubleshooting

**Problem:** Notifications not working
**Solution:** Click "Enable" on Reminders page and allow browser permissions

**Problem:** Data not saving
**Solution:** Check browser storage settings, ensure cookies/storage enabled

**Problem:** Charts not showing
**Solution:** Create more journal entries (need data for charts)

**Problem:** Achievements not unlocking
**Solution:** Achievements update when you visit the page, refresh if needed
